Tela: Cadastro Base SaaS

Visao geral
-----------
- Conjunto de telas placeholder para os cadastros estruturantes do SaaS.
- Separa os dominios base por rota dedicada: `Prioridade`, `Centro de Servico`, `Contrato`, `Imei`, `Tipo de Servico`, `Nivel de Tensao`, `Porte`, `Responsavel Distribuidora` e `Municipio`.
- Todas as rotas foram conectadas a permissao por pagina.

Arquitetura
-----------
- Rotas:
  - `src/app/(dashboard)/prioridade/page.tsx`
  - `src/app/(dashboard)/centro-servico/page.tsx`
  - `src/app/(dashboard)/contrato/page.tsx`
  - `src/app/(dashboard)/imei/page.tsx`
  - `src/app/(dashboard)/tipo-servico/page.tsx`
  - `src/app/(dashboard)/nivel-tensao/page.tsx`
  - `src/app/(dashboard)/porte/page.tsx`
  - `src/app/(dashboard)/responsavel-distribuidora/page.tsx`
  - `src/app/(dashboard)/municipio/page.tsx`
- Navegacao e guard de acesso: `src/components/layout/AppShell.tsx`
- Guard de rota por permissao: `src/lib/auth/authorization.ts`
- Catalogo de permissoes por usuario: `src/modules/dashboard/permissoes/PermissionsPageView.tsx`
- Base SQL de catalogo/permissao: `supabase/migrations/040_reorganize_menu_sections_and_page_permissions.sql`
