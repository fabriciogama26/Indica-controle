Edge Function: get_project_activity_forecast_template

Objetivo
- Baixar o modelo XLSX oficial para importacao de atividades previstas no projeto.
- Colunas do modelo: `projeto`, `codigo`, `quantidade`.

Secrets obrigatorios
- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY

Dependencias
- public.app_users com auth_user_id preenchido e ativo

Autenticacao
- Header `apikey` com `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- Header `Authorization: Bearer <access_token>`

Resposta
- Arquivo XLSX (`application/vnd.openxmlformats-officedocument.spreadsheetml.sheet`)
- Nome sugerido: `modelo_atividades_previstas.xlsx`
