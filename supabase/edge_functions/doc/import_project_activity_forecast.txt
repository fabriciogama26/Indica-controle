Edge Function: import_project_activity_forecast

Objetivo
- Importar planilha XLSX de atividades previstas para um ou varios projetos.
- Fluxo protegido por RPC antes da gravacao.

Secrets obrigatorios
- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY

Dependencias
- public.app_users com auth_user_id preenchido e ativo
- public.project
- public.service_activities
- public.project_activity_forecast
- RPC `precheck_project_activity_forecast_import`
- RPC `append_project_activity_forecast`

Autenticacao
- Header `apikey` com `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- Header `Authorization: Bearer <access_token>`

Payload
- `multipart/form-data`
  - `file`: XLSX com colunas `projeto`, `codigo` e `quantidade`

Protecoes de importacao
- Resolve `projeto` por `project.sob` dentro do tenant autenticado.
- Permite varios projetos no mesmo arquivo.
- Bloqueia codigo duplicado no mesmo projeto dentro do arquivo (`DUPLICATE_CODE_IN_FILE`).
- Ignora codigo ja importado para o projeto e continua cadastrando as linhas novas.
- Bloqueia codigo inexistente ou inativo no cadastro `service_activities` do tenant.
- Retorna `errorRows` para geracao de CSV de erros/avisos (`linha`, `coluna`, `valor`, `erro`).

Resposta de sucesso
- `success: true`
- `summary.rowsRead`
- `summary.projectsProcessed`
- `summary.activitiesRegistered`
- `summary.skippedRows`
- `summary.sourceFile`
