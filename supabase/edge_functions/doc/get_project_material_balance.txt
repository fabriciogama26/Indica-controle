Edge Function: get_project_material_balance

Objetivo
- Consultar o saldo liquido de materiais por projeto.

Secrets obrigatorios
- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY

Dependencias no banco
1. 012_create_project_material_balance.sql
2. public.app_users com auth_user_id preenchido

Autenticacao
- Exige Authorization: Bearer <access_token>
- Usa tenant_id do app_users vinculado ao auth.uid()

Payload aceito
{
  "projeto": "OMI-22-2122",
  "codigo": "210",
  "descricao": "capacitor",
  "qty_exact": 1,
  "qty_min": 1,
  "qty_max": 50,
  "limit": 100,
  "offset": 0
}

Observacoes
- Filtros sao opcionais.
- qty_exact, qty_min e qty_max filtram qty_net.
