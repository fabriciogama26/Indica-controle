Edge Function: get_inventory_balance

Objetivo
- Consultar o saldo fisico atual do estoque por material.

Secrets obrigatorios
- SUPABASE_URL
- SUPABASE_SERVICE_ROLE_KEY

Dependencias no banco
1. 001_create_core_tables.sql
2. public.app_users com auth_user_id preenchido

Autenticacao
- Exige Authorization: Bearer <access_token>
- Usa tenant_id do app_users vinculado ao auth.uid()

Payload aceito
{
  "codigo": "210",
  "descricao": "capacitor",
  "qty_exact": 10,
  "qty_min": 1,
  "qty_max": 50,
  "limit": 100,
  "offset": 0
}

Observacoes
- Filtros sao opcionais.
- qty_exact, qty_min e qty_max filtram qty_on_hand.
